# 📝  To-Do List

A modern and responsive **To-Do List Web Application** built using **HTML, CSS, JavaScript, Bootstrap, and Local Storage**. This project helps users organize their daily tasks with an intuitive interface and persistent data storage.

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Edit existing tasks
* 🗑️ Delete individual tasks
* ☑️ Mark tasks as completed
* 🔍 Search tasks instantly
* 🎯 Filter tasks (All / Completed / Pending)
* 📊 Task statistics (Total, Completed, Pending)
* 💾 Local Storage support (tasks remain after page refresh)
* 🌙 Dark Mode
* 📱 Fully Responsive Design
* 🎨 Modern UI with Bootstrap & Font Awesome
* ⚡ Press **Enter** to add a task
* 🗑️ Clear All Tasks option
